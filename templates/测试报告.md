## 三、测试内容

### 3.1 功能测试

| 编号 | 测试项 | 测试步骤 | 预期结果 | 实际结果 | 是否通过 |
|---|---|---|---|---|---|
| T1 | 系统上电初始化 | 上电后观察 OLED 和串口输出 | OLED 显示初始化信息，串口打印启动日志 | | |
| T2 | 传感器数据采集 | 用手改变温湿度或触发烟雾传感器 | OLED 显示数据变化，串口输出正确数值 | | |
| T3 | 本地显示功能 | 观察 OLED 屏幕 | 数据正确显示在指定位置 | | |
| T4 | 报警功能 | 使传感器数据超过阈值 | 蜂鸣器响起，OLED 显示报警信息 | | |
| T5 | Wi-Fi 连接 | 配置 Wi-Fi 后重启设备 | 串口显示连接成功，获取 IP 地址 | | |
| T6 | MQTT 连接 | 观察串口输出和云平台状态 | 设备在线，云平台显示在线状态 | | |
| T7 | 数据上报 | 观察云平台数据曲线 | 数据按设定周期成功上报 | | |
| T8 | 远程控制 | 从云平台下发控制指令 | 设备正确执行指令并反馈 | | |
| T9 | 系统稳定性 | 连续运行 30 分钟以上 | 无死机、无内存泄漏、数据持续正常 | | |
