# 测试计划

| 项目 | 内容 |
|---|---|
| 开发板 | Hi3861 |
| 开发工具 | DevEco Device Tool |
| 调试工具 | 串口助手，波特率 115200 |
| 传感器 | SHT30、烟雾传感器、可燃气体传感器 |

| 云平台 | OneNET / 腾讯云 IoT Explorer / 阿里云 IoT，最终以实际选择为准 |

## 二、测试目标

验证系统是否能够稳定完成环境数据采集、本地显示、异常报警和 MQTT 数据上报，确认系统满足课程设计的基本演示要求。

## 五、异常场景测试

| 编号 | 测试项 | 测试步骤 | 预期结果 |
|---|---|---|---|
| E1 | Wi-Fi 断开 | 关闭路由器或修改密码 | 系统不崩溃，本地显示和报警继续工作 |
| E2 | MQTT 断开 | 断开云平台连接 | 系统继续本地采集，串口提示重连 |
| E3 | 传感器异常 | 断开传感器连接或读取失败 | 串口输出错误信息，OLED 显示异常状态 |
| E4 | 数据越界 | 设置极低阈值触发报警 | 系统正确进入报警状态 |
