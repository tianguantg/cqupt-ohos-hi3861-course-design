# Bossay 开发板资料

本目录包含适用于 **Bossay 开发板** 的 HarmonyOS 设备开发教学资料。

## 文件清单

| 文件名 | 内容说明 |
|--------|----------|
| `第1章 C语言HarmonyOS设备开发环境.pptx` | 第一章课件：开发环境安装与配置 |
| `第2章 C语言HarmonyOS设备程序开发.pptx` | 第二章课件：设备端 C 语言程序开发（适用bossay开发板） |
| `第3章 C语言HarmonyOS设备程序开发方法、步骤.pptx` | 第三章课件：开发流程与调试方法 |
| `HarmonyOS设备开发环境搭建.pptx` | 补充课件：环境搭建专题讲解 |
| `实验用源码（适配bossay开发板）.docx` | 配套实验源码及说明 |

## 使用说明

### 课件（PPT）

按章节顺序学习（第1章 → 第2章 → 第3章），每章 PPT 包含：
- 理论知识讲解
- 操作步骤截图
- 代码示例
- 课后练习

### 实验源码（DOCX）

`实验用源码（适配bossay开发板）.docx` 包含可直接编译运行的示例代码。使用方式：

1. 用 Word / WPS 打开文档
2. 复制代码片段到 DevEco Device Tool 项目中
3. 按文档说明编译烧录

## 适合人群

- HarmonyOS / OpenHarmony 设备开发入门者
- 嵌入式 C 语言初学者
- 高校物联网/嵌入式专业课程教学

## 硬件要求

- Bossay 开发板（Hi3861 芯片）
- USB 数据线
- 可选：传感器扩展板
