风速逻辑 Speed Logic

AUTO 会先按温度决定基础目标风速，再根据湿度补偿，最后叠加 ECO/MORE/LESS 偏好。界面显示的是虚拟风速，不是电机持续转速。

电机默认常转：虚拟风速不为 0 时以最低功率持续转动；同时在等级变化或虚拟风速每变化 5% 时触发更敏感的短脉冲反馈。若现场电机转动噪音太大，可点击“关闭常转”切换为仅脉冲反馈。

OFF
目标 0%，立即回到 OFF。
QUIET
目标 25%；ECO 后为 15%。
AUTO Temp
温度基础目标：<26°C=0，26-27.4=30，27.5-28.9=45，29-30.9=60，≥31=75。
Humidity
湿度补偿：>65% 加 5%，>75% 加 10%。温度和湿度共同决定最终目标。
ECO / MORE / LESS
ECO 是 -10%；MORE/LESS 改变偏好偏置。
Smooth
每次采样最多变化 5%，所以目标没变或正在平滑逼近时，看起来会暂时不变。
Level
0=OFF，1-35=LOW，36-60=MEDIUM，61+=HIGH。
Motor Hold
ON：虚拟风速 > 0 时最低功率常转；OFF：关闭常转但保留 5% 风速变化脉冲反馈。
