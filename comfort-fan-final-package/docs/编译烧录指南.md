# 编译与烧录指南

## 1. 放置源码

OpenHarmony 根目录：

```bash
/home/ohos/workspace/openharmony-3.2-release
```

应用目录：

```bash
/home/ohos/workspace/openharmony-3.2-release/device/board/issedu/qihang/app/fan_manager
```

将 `openharmony_patch/` 中的文件复制到上述目录：

```bash
cp fan_manager.c "$ROOT/device/board/issedu/qihang/app/fan_manager/"
cp BUILD.gn "$ROOT/device/board/issedu/qihang/app/fan_manager/"
cp network_config.example.h "$ROOT/device/board/issedu/qihang/app/fan_manager/network_config.h"
```

然后编辑 `network_config.h`，填入实际 Wi-Fi 名称、密码和 MQTT Broker 地址。

## 2. 编译

```bash
export PATH="$HOME/.local/bin:$HOME/tools/gcc_riscv32/bin:$PATH"
ROOT="/home/ohos/workspace/openharmony-3.2-release"
cd "$ROOT"
hb build -f
```

输出固件：

```bash
$ROOT/out/qihang/qihang/Hi3861_wifiiot_app_allinone.bin
```

## 3. 复制固件到 Windows

```powershell
scp ohos@192.168.158.129:/home/ohos/workspace/openharmony-3.2-release/out/qihang/qihang/Hi3861_wifiiot_app_allinone.bin "D:\03_Assets\Exchange\OpenHarmony\qihang\"
```

## 4. 烧录

使用课程/开发板配套烧录工具，选择 allinone bin 烧录。烧录后打开串口观察：

```text
Wi-Fi link connected
DHCP ready
MQTT connected
```
